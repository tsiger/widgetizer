# Plugins

Plugins are a planned way to extend Widgetizer beyond what themes provide: custom integrations, editor tools, and functionality that isn't tied to a single theme's design.

This system is still in development and isn't available yet.

# Extending Widgetizer Today

Most of what you might reach for a plugin to do is already possible through **themes**. A theme can ship:

- **Custom widgets** with their own settings, markup, and behavior
- **Collections:** your own content types (News, Projects, Services) with item pages
- **Global settings** for colors, typography, and anything else you want users to control
- **Custom CSS and scripts** via theme settings and Liquid tags

See [Theme Development](theme-dev-quickstart.html) to build your own, starting from the quickstart.

We'll document the plugin API on this page once it's released.
